What is it?
===========

A Puppet module that ensures the linux audit packages are installed and
configured.  It has a basic set of rules provided, but can be overridden
with rules you provide.

Released under the Apache 2.0 licence

Usage:
------

To install:
<pre>
  include audit
</pre>
